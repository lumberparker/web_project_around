# Tripleten web_project_homeland

Descripción del Proyecto
Este proyecto es una página web interactiva donde los usuarios pueden ver y dar "me gusta" a fotos, así como editar la información de su perfil. Es una aplicación web responsiva que funciona en diversos tamaños de dispositivos.

Tecnologías Utilizadas:
HTML5
CSS3 (con metodología BEM)
JavaScript
Grid Layout
Flexbox

Características:
Diseño Responsivo: Funciona en dispositivos de escritorio y móviles
Edición de Perfil: Los usuarios pueden editar su nombre e información personal
Funcionalidad de "Me gusta": Los usuarios pueden dar y quitar "me gusta" a las fotos
Ventanas Modales: Experiencia limpia para la edición de perfiles
